```bash
git remote add origin https://github.com/VOTRE_USERNAME/gestion-commercial.git

```

### 🌐 Étape 2: Vercel (5 min)
```
1. Vercel.com → New Project
2. Importer dépôt GitHub
3. Build: cd frontend && npm run build
4. Output: frontend/dist
5. Ajouter: VITE_API_URL (mettre à jour après backend)
6. Déployer!
```

### 🔌 Étape 3: Render (10 min)
```
1. Render.com → New Service
2. Importer dépôt GitHub
3. Build: cd backend && npm install
4. Start: cd backend && npm start
5. Ajouter variables d'env (voir NEXT-STEPS.md)
6. Déployer!
```

### 🔗 Étape 4: Intégrer (2 min)
```
1. Vercel: Mettre à jour VITE_API_URL avec URL Render
2. Redéployer frontend
3. Tester!
```
